桃花源静态站
============

index.html 是桃花源的单页城镇指南，包含小镇介绍和桃花源法典两个主入口。
页面样式和交互分别由 assets/css/main.css、assets/js/taohuayuan.js 直接提供，不依赖第三方脚本。

内容约定：
- 当前 HTML 是线上发布内容的事实源；本轮依据桃花源提供的内容文档整理，原始文档只作为非公开编辑资料，不是构建依赖。
- 页面不再展示桃花源简史，也不在品牌或标题旁附加英文副标题。
- 法典正文保留六编二十七条，并把选举条件、适用情形和坐标附表转换为适合网页阅读的结构。
- 页面中的“整理日期”只说明网页内容整理时间，不代表法典生效日期或坐标核验日期。

阅读与交互：
- 桌面端使用章节侧栏，移动端使用粘性章节菜单；六编法典与坐标附表均有可分享锚点。
- 二十七条法典分别提供 article-1 至 article-27 的永久链接，滚动时同步提示当前主区块和章节。
- 交通图与 QQ 群二维码可点击查看原图；QQ群号同时提供可复制文本，图片加载失败时不影响加入信息获取。
- 无 JavaScript 时章节锚点、原图链接和群号文本仍可使用；键盘焦点与 reduced motion 均有对应降级。

图片策略：
- 页面优先加载 images/*.webp。
- 原始 JPG/PNG 作为源图保留，不删除、不覆盖；未被页面或主站内容引用时不进入生产包。
- 新增的交通示意图和 QQ 群二维码使用无损 WebP，避免小字号文字与二维码因有损压缩变糊，且比原始 PNG 合计减少约 48% 体积。
- 旧简史图片继续作为源资料保留；因 HTML 不再引用，生产构建不会发布。

本地检查时，从仓库根目录启动静态服务器后访问 /towns/taohuayuan/。
